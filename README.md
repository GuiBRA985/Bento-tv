# Bento TV

**O mundo cabe aqui.**

Site para explorar canais públicos da coleção IPTV-org, organizados por país e assunto.

## Arquivos

- `index.html`
- `style.css`
- `app.js`
- `CNAME`
- `.nojekyll`
- `assets/favicon.svg`

## Publicação

1. Envie todos os arquivos para a raiz do repositório `Bento-tv`.
2. No GitHub, abra **Settings → Pages**.
3. Escolha **Deploy from a branch**.
4. Selecione a branch principal (`master`) e a pasta `/root`.
5. Salve.

## DNS

Crie um registro CNAME:

- Nome: `tv`
- Destino: `GuiBRA985.github.io`

O arquivo `CNAME` já está configurado para `tv.bento.host`.

## Observação

O Bento TV não hospeda vídeos. O catálogo usa os links públicos da API do IPTV-org. Alguns canais podem estar temporariamente fora do ar, limitar países ou bloquear reprodução no navegador.
